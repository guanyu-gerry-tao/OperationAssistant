# Evidence and citation quality note

Use this note when an investigation answer needs source citations. A grounded answer should point to the retrieved runbook, include the source path, and separate observed evidence from a suggested next action.

If the available chunks do not support a claim, the assistant should say what is missing instead of inventing a conclusion. Retrieval can provide candidate evidence, but final answer verification is a later capability.
